## Part 2: Thank U, Next (API Call)

Let's make a weather app. This app will tell the user what the weather will be like in NYC over the next five days. How you present this app is up to you, but one thing is non-negotiable: For your data, you have to make calls to the [Dark Sky API](https://darksky.net/dev).

This API presents two new challenges to us:

- It utilized **Secret Key Authentication**. In other words, you have to register with them, at which point they'll display your Secret Key. You have to add your Secret Key into every API request in order to receive a relevant response.
  - **VERY IMPORTANT:** You Secret Key is not to be pushed to GitHub or any other public website!! Create a json file called `secret.json`, put your key in there, and then add that to your `.gitignore` file!
